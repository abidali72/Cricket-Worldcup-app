# Cricket World Cup App 🏏

A premium, feature-rich Android application designed for cricket enthusiasts to stay updated with the World Cup. Built with modern Android development practices using Kotlin and Jetpack Compose.

## ✨ Features

- **Live Match Updates**: Real-time scores and match details.
- **Points Table**: Keep track of team standings and statistics.
- **Live Streaming**: Integrated video player for live match broadcasts.
- **Match Predictions**: Advanced algorithms to predict match winners.
- **Futuristic UI**: Immersive design with glassmorphism and smooth animations.

## 🚀 Tech Stack

- **Language**: [Kotlin](https://kotlinlang.org/)
- **UI Framework**: [Jetpack Compose](https://developer.android.com/jetpack/compose)
- **Architecture**: MVVM (Model-View-ViewModel)
- **Dependency Injection**: Hilt (optional/assumed)
- **Networking**: Retrofit / OkHttp
- **Video Playback**: ExoPlayer / Media3

## 🛠️ Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/abidali72/Cricket-Worldcup-app.git
   ```
2. Open the project in **Android Studio (Ladybug or later)**.
3. Sync Project with Gradle Files.
4. Run the app on an emulator or physical device.
